# 📝 To-Do Web Application

A responsive and user-friendly To-Do Web Application built using **HTML, CSS, and JavaScript**. This application helps users organize their daily tasks by allowing them to add, complete, delete, and manage tasks efficiently. All tasks are automatically saved using the browser's Local Storage.

---

## 📌 Project Overview

The To-Do Web Application is designed to improve productivity by providing a simple and intuitive interface for managing daily tasks. It demonstrates core JavaScript concepts such as DOM manipulation, event handling, Local Storage, and dynamic content rendering.

---

## ✨ Features

- 🏠 Welcome Screen
- ➕ Add New Tasks
- ✅ Mark Tasks as Completed
- 🗑️ Delete Individual Tasks
- 📊 Task Statistics (Total, Completed, Pending)
- 🧹 Clear All Tasks
- 💾 Automatic Local Storage
- ⌨️ Add Tasks by Pressing Enter
- 📱 Fully Responsive Design
- 🎨 Modern and Clean User Interface
